# Feature Surface

This document records the capability surface found in the earlier vibecoded repository.

It is historical evidence, not a starting point for the new implementation.

## Purpose

Use this file when you need to answer:

- what the earlier system implemented
- which features were user-visible
- which internal capabilities existed
- which data structures and integrations were present

## Summary

The earlier system included:

- an interactive drawing canvas
- persistent places and explicit lines
- circular fields, axes, bends, and circular repeaters
- staged commit/reject editing
- transformation history
- structural inspection
- zoom, pan, and gesture support
